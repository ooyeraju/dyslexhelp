Hey Teachers! Here's what you have to do...

This section is to generate the images which the kids will read for practice. Please go through the instrctions very carefully


Here's what you have to do:

  • If you go into the lists directory, you can see a new directory named {{ x }}. Two files will be present inside it. One will be of .txt format and the other will be of .csv format. Don't do anything with the .csv file
  • Click on the PROCEED button
  • You'll be directed to a new website in another tab.
  • Copy the whole content of the text file and then, paste it in the area provided below (it has a blue color 'Generate image' button above it). Make sure you remove all the previous contents in the box.
  • Then, click on Generate Image. Make the name of the image {{ x }}.jpeg. Put the renamed image in the './static/img' directory.
  • Bingo! You're done.
  • Make sure you rename the image properly and put it in the correct directory.
  • We have provided you with 4 example cases.
  • It would be great if you could explain the kids what they have to do in the reading section.
